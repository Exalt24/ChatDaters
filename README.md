# ChatDaters

Dating platform with Tinder-style swipe matching, real-time match notifications, and in-app messaging. Built solo in a 3-day sprint during my Full Stack Developer internship at ChatGenie.PH (Jul 2024).

- `backend/` - Ruby on Rails API (WebSocket via Action Cable)
- `frontend/` - Vue 3 + Vite + Tailwind CSS

Merged from the original ChatDaters (backend) and ChatDatersFrontEnd repos with full history preserved.
